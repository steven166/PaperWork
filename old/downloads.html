First use

  1. This bootstrap requires JQuery

    <script src="jquery.js"></script>

  2. Download Material Design bootstrap. It contains a css and a javascript file. Add both files to your wesite and add the following code to your html page header:

    <link rel="stylesheet" href="materials.css"/>
    <script src="materials.js"></script>

  3. No you are ready to use the Material Design bootstrap. See Paper Elements for more details and documentation
